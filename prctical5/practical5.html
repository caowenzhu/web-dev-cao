a. What is a box model in CSS. Give box-sizing property definition

The box model in CSS is the concept of elements taking up space in a page layout. Each HTML element is represented as a rectangular box on the page, occupies a certain space and can express meaning.

The box-sizing attribute can set the size calculation rules for the element box. It has two possible values:

1. content-box: When setting the width and height of an element, it will only be applied to the content area.

2. border-box: When setting the width and height of an element, these dimensions will be applied to the entire element box.

b. What is a “clearfix hack” and when does it need to be applied

ClearFix is a way to have elements automatically clear or fix their elements so that no additional markup is required.

When you want to solve a height collapse problem using pure CSS, you can use the "clearfix hack" to achieve this. or, when floating elements cause layout clutter,also can use “clearfix hack” to solve the problem.

c. What is a “margin collapse” and what are the use cases for margin: auto and negative margin values

"Margin collapse" occurs when the top and bottom margins of two adjacent elements are combined (collapsed) into a single margin.

auto: This automatically centers the element and divides the space equally. For example css .center( Width: 200px; Margin: automatic; )

Negative margin values: Negative margins can be used to create an overlapping effect, adjust positioning, or bring elements closer to each other.

d. What was the problem with achieving Holy Grail layout

Same Height Columns: Make the header, main content, and footer the same height, which is difficult to achieve using just CSS without resorting to hacks or workarounds.

Fluid and Responsive Design: Holy Grail layout becomes fluid and responsive in order to adapt to different screen sizes and devices.

e. What are the modern ways to achieve Holy Grail layout
